Тестирование Яндекс.Самоката (https://qa-scooter.praktikum-services.ru/)
«В рамках данного проекта была выполнена комплексная автоматизация пользовательских сценариев веб-интерфейса сервиса доставки. Решение включало три ключевых направления тестирования:

Валидацию механизма раскрытия контента в разделе FAQ — с отдельной проверкой каждого вопроса.
Полное покрытие позитивного сценария оформления заказа, при этом вариативность данных была обеспечена за счёт параметризации.
Верификацию корректности навигационных элементов — в частности, переходов по логотипам на целевые страницы.
Для обеспечения масштабируемости и поддержки кода была применена архитектурная модель Page Object: страницы были чётко разделены на отдельные классы, а локаторы вынесены в обособленные модули. Результаты выполнения тестов визуализированы с помощью детального Allure-отчёта.

Опыт реализации проекта подтвердил значимость проработки архитектуры на этапе планирования — это существенно упрощает дальнейшее расширение и поддержку тестового набора.»
